问题3

执行：/usr/sbin/mysqld --initialize-insecure --user=mysql

  1. 执行：yum install -y libaio
  2. 然后执行：/usr/sbin/mysqld –initialize-insecure –user=mysql

问题4

  1. 查看日志：tail -n 20 /var/log/mysqld.log
  1. 通过yum安装的话
  2. 5.7版本以前是 chmod -R 777 /usr/local/mysql/data/
  3. 5.7版本以后是 chmod -R 777 /var/lib/mysql
  4. 如果不是通过yum安装的话：find / -name ibdata1
  5. chmod -R 777 /var/lib/mysql/ibdata1
