python命名规范

发表于 2019-08-19 分类于 python 评论数： 阅读次数：

标识符

常量，变量就是标识符的一种

变量命名规则

  1. 不能以数字开头
  2. 只能包含字母、数字和下划线
  3. 区分字母大小写
  4. 禁止使用保留字
  5. 见名知意

命名惯例

  • 以单一下划线开头变量名（_x）不会被from module import * 语句导入，是私有的
  • 前后有下划线的变量名（x）是系统定义的变量名，对python解释器有特殊意义
  • 以两个下划线开头但结尾没有下划线的变量名（__x）是类的本地变量
  • 交互式模式下，变量名“_”用于保存最后表达式的结果
  • 变量名没有类型，对象才有
  • 一般全大写的为常量
