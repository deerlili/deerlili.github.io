awk统计ip访问

发表于 2019-08-03 分类于 linux

统计IP访问情况并分析访问日志

日志

  1. 统计出每个 IP 的访问量有多少

解析命令

  1. awk 命令在分段方面还是比较有优势的，这里的{print $1}讲第一段打印出来，awk 可以用-F 指定分隔符，如果不指定分隔符，默认就以空白字符(比如空格、Tab 等)，本题中，IP 地址就是在第一段。
  2. sort 命令是排序的命令，-n 选项表示以数字的形式排序，如果不加-n，则以 ASCII 排序，本题中的 IP 地址以数字的形式排序更容易区分。
  3. uniq 命令是用来去重复的，一个文本中如果有多行内容是一模一样的，使用 uniq 命令就可以把相同内容的行给删除掉，只留一行。而-c 选项的作用是计算重复的行数，所以在此题中使用 uniq -c 正好可以计算 IP 地址的访问数量。不过，大家一定要注意，uniq 去重的前提是首先要排序。
  4. 最后没得 sort -n 意思是按访问量大小来排序，请求量越大的 IP 排在越后面，如果要想排在前面，可以加一个-r 选项，即 sort –nr

sqoop2错误

发表于 2019-07-30 分类于 错误

问题1

启动sqoop2 server
Caused by: ERROR XSDB6: Another instance of Derby may have already booted the database /var/lib/sqoop2/repository/db

解决办法：

  1. 查找进程：ps -aux|grep sqoop2
  2. kill进程号：kill -9 ...
  3. 重新启动

问题2

/usr/bdp/1.0.0-1/sqoop2/bin/sqoop.sh: line 202: /var/run/sqoop2/sqoop-sqoop2-jetty-server.pid: No such file or directory ERROR: Cannot write pid

解决办法：

  1. 在安装server的机器上建目录和文件
  2. cd /var/run/
  3. mkdir sqoop2
  4. chown sqoop2:hadoop sqoop2
  5. cd sqoop2
  6. touch sqoop-sqoop2-jetty-server.pid
  7. chown sqoop2:hadoop *
  8. 重启

python字符串

发表于 2019-07-30 分类于 python

字符串的申明：单引号,双引号，三引号

  • s = 'Hello Python!'
  • s = "Hello Python!"
  • s = '''Hello Python!'''
  • s = """Hello Python!"""
  • 字符串中有单引号用双引号包裹，有双引号用单引号包裹，有换行用三引号（单引号，双引号）包裹

    字符串的操作

  1. 单个访问字符串中的字符
  • s = 'Hello Python!'
  • print(s[0]) 输出为：H
  1. 访问字符串的子串（切片操作）
  • print(s[0:5]) 输出为：Hello
  1. 字符串相加
  • s1 = 'Hello'
  • s2 = ' Python!'
  • print(s1+s2) 输出为：Hello Python!
  1. 字符串更新操作
  • s1 = 'Hello'
  • s2 = 'Python'
  • print(s1[:2]+s2) 输出为：HePython
  1. 字符串的成员运输
  • 包含运算：in，不包含：not in
  • s1 = 'Hello'
  • s2 = 'H'
  • print(s2 in s1) 输出为：True
  1. 转义字符（换行符号：\n 制表符：\t 回车：\r 这个只打印\r之后的）
  • print("\"")
  1. 原始字符串
  • print(r"hello\nPython")
  • print(R"hello\nPython")
  1. 字符串的格式化输出
  • print("我是%s",第%d天学习" %('小明',10))
  1. 字符串的内建函数
  • 查找字符串
  • s = "Hello Python".find("He") 范围位置
  • 转换为大小写字母
  • s = "Hello Python".upper()
  • s = "Hello Python".lower()
  • 字符串的长度
  • s = "Hello Python".__len__()
  • 判断字符串是否只包含空格
  • print("a ".isspace())
  • 字符串替换
  • print("Hello Python".replace("o","ee"))
  • 输出20次
  • print("he"*20)
